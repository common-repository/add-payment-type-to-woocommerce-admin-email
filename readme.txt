== Description ==

Add Payment Type to WooCommerce Admin E-Mail plugin adds a functionality to Woocommerce plugin that displays in admin e-mail payment method. This plugin requires Woocommerce.


== Installation ==

Upload the Add Payment Type to WooCommerce Admin E-Mail plugin to your blog, activate it and this it.
